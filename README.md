# Requirements
    - You need to have Smooch account
    - You need to generate `jwt` token. I am using `pyjwt` to generate.
    - You need to have Vision API Key

# Setup
    - Make `config.ini`. The config file is placed in the same level as `main.py`
    - Fill in the details.

# `config.ini` template
```ini
[keys]
api_key = xxx
jwt_token = xxx
```
